# Dokumentasi

### Migration
```php artisan migrate```

### Seeder
```php artisan db:seed```

### Endpoint

| Method | URL |
| ------ | ------ |
| GET | /getallitems |
| POST | /searchitem |
| GET | /item/{code} |
| POST | /item/ |
| PUT | /item/{code} |
| DELETE | /item/{code} |
